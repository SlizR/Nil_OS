### 2. Live System Data Isolation and Private Territory

A fundamental feature of Nil OS, powered by the **EVADE Security Service**, is the absolute isolation of user changes and data from the host computer's internal storage.

**Core Isolation Principle:**

> **When Nil OS is run as a Live system from a removable drive (such as a USB flash drive), all user modifications, configuration changes, and saved data are written exclusively to the flash drive itself.**

**Resulting Guarantee:**

> **System Isolation:** "I inserted a flash drive with the operating system (Nil OS) into the computer, made changes, and all the data was saved to the flash drive, and nothing changed or was saved on the PC."

This creates a **"Private Territory"** on the flash drive, ensuring user privacy and protecting the integrity of the host machine's operating system and hard drive.
